Exercice 3

Avec le formulaire de l'exercice 1, informez l’utilisateur de la sécurité du « mot de passe ». Pour que l’UX soit optimum : le faire en temps réel.

  • Faible : Le point de départ, l’utilisateur saisit quelque chose.
  • Faible : 8 caractères minimum + une majuscule.
  • Moyen : 8 caractère minimum + une majuscule + chiffre.
  • Fort : 8 caractère minimum + une majuscule + chiffre + caractère spécial.
